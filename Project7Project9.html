Project 7: Data Exploration with Pandas

About the Project

This project focused on exploring a real-world dataset of electric vehicles registered in Washington State. Using Python libraries like Pandas, Matplotlib, and Seaborn, I performed data cleaning, aggregation, and visualization to uncover insights about electric vehicle types, registration trends, and county-level adoption. Like project 6, this project allowed me to further understand data analysis via python, utilizing real-world examples.
